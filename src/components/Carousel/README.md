# Carousel

| Prop        | Type    | Description                                    |
| ----------- | ------- | ---------------------------------------------- |
| `loop?`     | boolean | ループさせる                                   |
| `auto?`     | boolean | 自動でスライドさせる                           |
| `interval?` | number  | `auto`が`true`の場合何秒ごとにスライドさせるか |
| `length`    | number  | スライドの合計                                 |
| `show?`     | number  | スライドの前後に必要なスライドの数             |
| `swipe?`    | boolean | マウス、タッチでの操作を許可する               |
